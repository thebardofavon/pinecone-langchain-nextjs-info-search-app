##### Instructions to Candidate {#irut4 .text-center .text-success draggable="true"}
:::

::: {#imdqy .card-body draggable="true"}

-   [There is no application fee.]{#i5znf}
-   [Keep ready the soft copy of the following documents (if applicable)
    for uploading:]{#ibmqk}
    1.  [All revelant certificates and grade-sheets, as per the academic
        qualification.]{#itpsk}
    2.  [Valid GATE/NET scorecard (if applicable).]{#irq9t}
    3.  [Category certificate (if applicable).]{#ikkva}
    4.  [PwD certificate (if applicable).]{#ii56j}
    5.  [Professional experience proof (if applicable).]{#iv3qh}
    6.  [Arrange the above listed documents in order and prepare a
        single PDF file. Please ensure that the file size of the single
        PDF file is not exceeding 10MB.]{#igjya}

##### Application Process {#if7tn .text-center .text-success draggable="true"}
:::

::: {#idn8g .card-body draggable="true"}
Interested candidates are requested to apply by filling up the Google
Form in the link given below:

[Apply Now (Click
here)](https://docs.google.com/forms/d/e/1FAIpQLSdb-g7fPwqOhY-ytU8Xs5702EgVvAK6CcnEGnKD8i2e0H8nAw/viewform){#islr1
.btn .btn-primary draggable="true" target="_blank"}

Application form along with the self-attested copies of all degrees;
mark sheets and GATE / NET score card (if applicable) copy must be
uploaded. The application form (other than Annexures) should be filled
in the provided Word File template and converted to pdf before
uploading.

**Download the Application Form**

[Application
Form](https://www.iiitg.ac.in/uploads/2023/10/07/d4d74c22f8ba905bfbbd3a5ea8228b78.docx){#itgcw
.btn .btn-primary draggable="true" target="_blank"}

**Selection Procedure**

Shortlisted candidates will be called for an interview through email.
The interviews would be via video conferencing.
